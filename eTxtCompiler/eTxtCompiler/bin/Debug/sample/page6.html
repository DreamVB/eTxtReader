Options

The options displayed in the image allow you to preform different actions I. They will be detailed below

  • Generate

    Use this to generate the random passwords using the steps shown on the last pages.

  • Select All

    Highlights and selects all the generated passwords.

  • Copy Selected

    Copy all the selected passwords to the clipboard for latter use.

  • Save

    Takes the passwords from the list and allows saving to an external file.

  • About

    Just displays information about this application.
